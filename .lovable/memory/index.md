Landing page for dot — premium café/events brand marketing site. Beige, blue, orange palette.

## Design System
- Font display: Outfit (Halis Grotesque direction, installed via @fontsource/outfit)
- Font body: DM Sans
- Background: warm beige HSL 37 26% 89%
- Card: off-white HSL 40 11% 96%
- Primary blue: HSL 218 88% 50%
- Secondary blue: HSL 216 62% 69%
- Accent orange: HSL 14 89% 54%
- Foreground: #111111
- Muted foreground: #333333
- Use `font-display` for headings, `font-body` for body text

## Architecture
- Landing page only (no admin, no portal)
- All data hardcoded in src/data/landing.ts
- Components in src/components/landing/
- Brand-forward marketing tone, not franchise portal tone
